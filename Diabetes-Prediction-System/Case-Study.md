# Case Study: Diabetes Prediction System

## Problem Statement

Diabetes is one of the most common chronic diseases worldwide. Early detection can help patients take preventive measures and reduce health risks.

The objective of this project was to develop a Machine Learning model capable of predicting diabetes based on patient medical information.

---

## Business Objective

Healthcare organizations and clinics often require quick risk assessments for diabetes.

This system provides an automated prediction mechanism that can assist healthcare professionals in identifying high-risk individuals.

---

## Dataset

The project uses the Pima Indians Diabetes Dataset.

Key attributes include:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

Target Variable:

* Outcome (0 = Non-Diabetic)
* Outcome (1 = Diabetic)

---

## Data Preprocessing

The following preprocessing steps were performed:

* Data Cleaning
* Feature Selection
* Standardization using StandardScaler
* Train-Test Split

---

## Machine Learning Model

Algorithm Used:

Support Vector Machine (SVM)

Reasons for Selection:

* Effective for classification tasks
* Works well on small-to-medium datasets
* Good generalization capability

---

## Model Development Process

1. Data Loading
2. Exploratory Data Analysis
3. Feature Scaling
4. Train-Test Split
5. SVM Model Training
6. Model Evaluation
7. Model Saving using Pickle
8. Deployment through Web Application

---

## Results

The trained model successfully predicts whether a patient is diabetic based on medical measurements.

The project demonstrates the complete Machine Learning lifecycle from data preparation to deployment.
